# Contatos

Programa que simula uma lista telefonica

# Funcionalidades

- Visualizar a Lista
- Buscar Contato
- Adicionar Contato
- Atualizar Contato
- Deletar Contato

# Tecnologias

JAVA

# Explicação

Usei a Arraylist pela gama de beneficios que ela poderia me trazer como metodos praticamente prontos, dinamica e lista com ordem de inserção.
